# Homebrew Tap

Official Homebrew tap for projects by [Arjun Komath](https://github.com/arjunkomath).

## Usage

```bash
brew tap arjunkomath/tap
```

Then install any available formula or cask:

```bash
brew install <formula>
brew install --cask <cask>
```
